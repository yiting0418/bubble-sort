# 氣泡排序法 (Bubble Sort)

Python 實現的氣泡排序演算法。

## 使用方式

```python
from bubble_sort import bubble_sort

numbers = [64, 34, 25, 12, 22, 11, 90]
sorted_numbers = bubble_sort(numbers)
print(sorted_numbers)  # [11, 12, 22, 25, 34, 64, 90]
```

## 演算法特性

- 時間複雜度：O(n²)
- 空間複雜度：O(1)
- 穩定排序：是

## 執行

```bash
python bubble_sort/bubble_sort.py
```
